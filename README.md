# OE-Encoder Dataset Visualizer

Random samples + labels from 5 downloaded datasets:

- DeepFashion2 (191K + 32K clothing images with bbox + category)
- LogoDet-3K (3K brand classes with bbox + industry)
- VisualNews (1.26M news image + caption pairs)
- Herbarium 2020 / 2022 (FGVC plant specimen images with scientific name + taxonomy)
- Open Food Facts (food product photos + barcode-linked metadata: name, ingredients, Nutri-Score, etc.)

Live page: https://hellomuffin.github.io/oeencoder-dataset-viz/

Built with `tools/build_dataset_viz.py` from `~/open_clip`.
